Get Weather & Forecast App

About

The Get Weather & Forecast Application is a website that uses Weather API. The information granted to users within this app is much more extensive than the data on your average weather app that comes pre-installed on mobile devices. With the Get Weather & Forecast Application users can gather a plethora of details such as wind speeds, UV indexes, humidity levels and much more! Currently this website does not possess user login capabilities, but there are plans to add this feature in the future.
